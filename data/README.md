# Data Sources

This folder contains deterministic synthetic data for a streaming product analytics workflow. The data is generated by `scripts/generate_streaming_artifact.py` with a fixed random seed.

The data is modeled on common digital entertainment analytics structures:

- Feature metadata across home, discovery, live, profile, playback, ads, search, social, and notification surfaces.
- Daily metric mart rows by feature, device family, service tier, and customer segment.
- Experiment result rows with control and treatment variants, A/B tests, multivariate tests, primary metric outcomes, and playback guardrails.
- Metric definition records that document source tables, owners, and metric roles.
- Data quality checks for freshness, completeness, duplicate exposure, metric definition, and segment joins.
- Stakeholder requests that represent ad hoc product and business questions.

Generated source tables:

- `product_features.csv`
- `daily_feature_metrics.csv`
- `experiment_results.csv`
- `metric_definitions.csv`
- `data_quality_checks.csv`
- `stakeholder_requests.csv`

Generated analysis outputs:

- `analysis/outputs/product_priority_queue.csv`
- `analysis/outputs/experiment_readouts.csv`
- `analysis/outputs/segment_summary.csv`
- `analysis/outputs/metric_trust_queue.csv`
- `analysis/outputs/summary.json`
